# Context Store

The context store is a read-only knowledge base of user preferences and facts.

## What It Contains

Context entries are markdown files organized by topic. Examples:
- Food preferences and dietary restrictions
- Daily routines and schedules
- Household information
- Personal preferences

## How It's Used

When the chatbot responds to your messages, it searches the context store for relevant entries and uses them to personalize its responses. For example, if you ask about dinner ideas, it might reference your food preferences.

## How to Update It

Context entries are managed by the system administrator through markdown files. Ask your admin to add or update your preferences if needed.
